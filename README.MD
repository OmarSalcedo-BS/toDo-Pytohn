
# Momento 1 - Python

## Descripción del Proyecto
Este es el primer entregable del curso de Python, que contiene ejercicios y conceptos fundamentales del lenguaje.

## Propósito
Aplicar conocimientos básicos de Python mediante la práctica de programación estructurada y comprensión de conceptos clave.

## Ramas del Proyecto

### 1. **main**
Rama principal que contiene la versión estable y revisada del código. Solo se integran cambios probados y aprobados.

### 2. **dev**
Rama de desarrollo donde se trabajan nuevas funcionalidades y se hacen pruebas. Esta es la rama de integración antes de pasar a `main`.

### 3. **feature/nuevas_funciones**
Rama para características específicas. Se crea desde `dev` y se fusiona nuevamente a `dev` una vez completada la funcionalidad.

## Flujo de Trabajo
1. Crear rama desde `dev` para trabajar en nuevas características
2. Hacer commits en la rama de feature
3. Integrar cambios a `dev`
4. Una vez validado, integrar a `main`
